Important : Désactivez l'adresse MAC aléatoire

Pour que le pointage fonctionne, votre appareil doit utiliser son adresse MAC réelle sur le réseau Wi‑Fi "EMES".

  • iPhone/iPad : Réglages → Wi‑Fi → (i) à côté de "EMES" → Désactiver "Adresse privée".
  • Android : Paramètres → Wi‑Fi → ⚙️ (roue dentée) → "Type d'adresse MAC" → "Adresse MAC du téléphone".
  • Windows / macOS : L'adresse MAC est généralement fixe.
